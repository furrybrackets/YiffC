# YiffC
A high-level IR language designed to be a frontend for [LibFirm](https://github.com/libfirm/libfirm), since LLVM is shitty.


## Running YiffC from the command-line

First, build Yiff with the `cli` option. Then, run:

```
yiff $(file)
```
